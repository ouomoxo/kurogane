# RECOVERY_PLAN — Arasaka Experience Authenticity Program

Owner reset directive (2026-07-11). This is a MULTI-DAY iterative program, not
a checklist to clear in one pass. **Rules for every cycle:**

- A phase may NOT be marked complete in the same cycle it was started.
- Every iteration must produce rendered before/after evidence (Playwright,
  real Chrome), inspected and logged in SESSION_LOG.
- Scores require written evidence per criterion; no inflation; when in doubt,
  score lower and add a correction item.
- "Steady state" is forbidden until every phase below is closed AND the owner
  signs off. Health-check-only cycles are not permitted while this plan is open.
- Atmosphere > asset count. If simpler reads more Arasaka, simplify.

## P1 — Direction matrix, staged (directive §10.5–8) — OPEN
The earlier /proto comparison was void-staged, single-frame. Insufficient.
Rebuild A (Corporate Reliquary) and B (Sovereign Defense Core) INSIDE the
machine-space hall (reflective floor, perimeter columns, glyph), then render
ALL THREE directions at: desktop hero · 25% scroll · 50% scroll · transition

evidence. Compare; confirm or overturn the Data Temple choice. Minimum 3 cycles.

## P2 — Winner polish loop: silhouette & material (directive §6) — OPEN
Iterate the chosen core object: controlled asymmetry pass (currently fully
symmetric = weakness), manufacturing logic (how would this be assembled —
panel joins, service hatches, lift points), scale cues (small elements that
imply hugeness), reflection composition on the floor, ceramic/glass balance,
red-seam placement logic (every seam must mean something). Minimum 3 cycles,
each with close-up + full-frame renders.

## P3 — Motion direction (directive §8) — OPEN
- Ignition timing curves tuned against video capture (Playwright screencast).
- Scroll phase map authored as one timeline: title yield → strata open →
  camera push → Record 001 arrival; eliminate dead zones; verify with
  frame-stepped captures at 10% increments.
- Threshold transitions BETWEEN records (crossing system layers, not
  scrolling a page).
- Surveillance micro-cues: section-entry acknowledgment stamps, idle
  observation line (elegant, unsettling, no gimmick). Minimum 3 cycles.

## P4 — Typography/UI ↔ scene integration (directive §7) — OPEN
Label placement logic map (why each label sits where it sits), classification
bar tied to scene state, headline/scene composition audited at 2560/1920/
1440/1024/430/390/360, spatial labels behavior on scroll. Minimum 2 cycles.

## P5 — Institutional interface behavior (directive §4) — OPEN
System-state transitions when navigating divisions (crossing thresholds, not
loading pages); division pages restaged in the hall language with state-driven
seams (colonnade/archive first per ASSET_AUDIT); biometric/clearance flavor in
the contact/careers surfaces. Minimum 3 cycles.

## P6 — Final art-direction audit (directive Phase 10) — BLOCKED until P1–P5
Three full-site loops: (A) composition/typography/scale/brand,
(B) materials/lighting/motion/transitions, (C) reduction/consistency/
responsive. Each loop = rendered evidence + concrete fixes. Minimum 3 cycles.
