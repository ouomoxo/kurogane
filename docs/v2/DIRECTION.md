## 감독 규칙 (내가 정한다)
1. **v1은 폐기, 기록은 보존.** 태그 `v1-arasaka-archive`. 새 작품은 백지에서.
2. **선(先)작품, 후(後)코드.** 연출·시나리오·디자인이 합의한 크리에이티브 바이블
   없이는 프로덕션 코드를 쓰지 않는다.
3. **역할은 실제 서브에이전트 세션으로 분리한다.** 각자 독립 산출물을 내고,
   합동 평가(전원 + 감독)에서 서로의 것을 비평한다. 감독은 종합하고 결정한다.
4. **정직 게이트 유지.** 렌더 증거 없는 완성 주장 금지. 판정은 렌더로만.
5. **법적 가드레일 유지.** 팬 컨셉 — CDPR/R.Talsorian 에셋·로고·음원 미사용,
   오리지널 콘텐츠만, 고지 유지.
6. **미학 강령(초안, 바이블에서 확정):** 검정을 재질로, 적색을 신호로.
   네온 클리셰·보라 그라데이션·글리치 남용 금지. 시네마틱 = 카메라와 편집의
   언어(컷, 홀드, 리빌)를 웹 문법으로 번역한 것.
