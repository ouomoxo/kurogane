# ROLES — 역할 브리프 (합동 평가 프로토콜 포함)

공통: 주제 사이버펑크 2077 · 블랙/적색 · 오리지널 콘텐츠만(법적 가드레일) ·
매체는 웹(데스크톱+모바일) · 서두르지 않는다 · 거짓 없는 자기평가.

## 1 영화 연출 감독 (Cinematography)
산출물: 시네마틱 트리트먼트 — 오프닝 20초의 샷 리스트, 카메라 언어(웹 문법
번역: 스크롤=달리? 컷=섹션 전환?), 조명 설계(블랙/레드), 리듬/편집 원칙,
레퍼런스 원리(모방 아님) 5개.

## 2 시나리오 감독 (Scenario)
산출물: 내러티브 컨셉 — 방문자는 누구이며 무엇을 겪는가, 이야기 구조(막/장),
목소리와 톤(영/일 병용 여부), 핵심 카피 12줄 초안, 세계관 준수 노트
(2077 lore와 충돌하지 않되 복제하지 않는 오리지널 스토리).

## 3 전문 디자이너 (Design)
산출물: 비주얼 언어 제안 — 타이포 시스템, 블랙/레드 팔레트의 계층, 레이아웃
문법, 모션 원칙, UI 밀도(2077의 HUD 과밀을 어떻게 절제로 번역할지),
무드보드 서술(이미지 없이 원리로).

## 4 개발자 (Engineering)
산출물: 기술 접근서 — 위 셋을 구현할 스택/기법(현 RR7 인프라 재사용 여부 포함),
성능 전략, 모션 구현 방식(CSS/WAAPI/GSAP/canvas 선택 근거), 리스크 목록.

## 합동 평가 (S2, 매 리뷰 공통)
- 전원이 타인의 산출물을 각자 비평: 강점 2 · 결함 2 · 채택 권고(채택/수정/기각).
- 감독이 전 비평을 종합해 결정문 작성. 동점/충돌 시 감독 결정이 최종.
- 모든 평가는 구체 근거 필수. "좋다/약하다" 단독 서술 금지.
