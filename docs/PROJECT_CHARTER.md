# PROJECT_CHARTER — PROJECT KUROGANE

## Purpose
An in-universe corporate experience for the fictional Arasaka Corporation. The
site is a demonstration of premium interactive craft: it speaks entirely in
Arasaka's own institutional voice — calm, sovereign, quietly ominous — and never
breaks character except in the legally-required disclaimer.

## Success criteria
- Reads as a real, expensive corporate site, not a fan tribute or "cyberpunk skin".
- Every route carries genuine, distinct copy — no lorem, no placeholder.
- Cinematic 3D hero that is procedural, offline-capable, and performant.
- Passes the quality rubric: no category below 8; art/composition/typography ≥ 9.
- Ships legally clean: original/procedural assets only, disclaimer in footer + /legal.

## Non-goals
- Not a game, not a store, not a real corporation.
- No neon-soaked cliché cyberpunk. Restraint over spectacle.
- No login, no backend, no data collection.

## Constraints
- Static-hostable on GitHub Pages (no server).
- Fictional-concept legal posture: unaffiliated with CD PROJEKT RED / R. Talsorian.
